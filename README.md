3.)  Course Allocation System (Python) 

## Overview
This project implements a course assignment system using Python and pandas to simulate how students are allocated to courses based on preferences and constraints.

## Features
- Data cleaning and preprocessing (handling missing values and duplicates)
- Course allocation based on student preferences and percentile ranking
- Multi-wave assignment system (first, second, and third waves)
- Capacity management for each course
- Validation using assertions to ensure correctness
- Export of final results to CSV files

## Technologies Used
- Python
- pandas

## How It Works
1. Clean and preprocess student data
2. Determine how many courses each student must take
3. Assign courses based on priority (fall_1, fall_2, fall_3)
4. Respect course capacity limits
5. Validate results using automated checks
6. Generate final assignment outputs

## What I Learned
- Data preprocessing and transformation with pandas
- Designing algorithms with constraints
- Handling real-world scenarios like capacity limits
- Debugging and validating results using assertions
